# Lumina — Minimal studio website

A refined, modern single-page website template for studios, agencies, and personal brands. Built with a clean black-and-white design system, smooth Lenis scrolling, and subtle GSAP-driven interactions.

## What's included
- `index.html` — standalone site with loader, smooth scroll, and reveal animations
- `Lumina.tsx` — ready-to-use React component that renders the same design
- Accessible markup, responsive layout, and light-touch motion
- MIT licensed

## Quick start
- Open `index.html` directly
- Or import `Lumina` in a Next.js / React app

## Tech
- HTML + Tailwind CSS via CDN
- GSAP + ScrollTrigger
- Lenis smooth scroll
