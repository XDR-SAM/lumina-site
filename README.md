# Lumina — Minimal studio website

A refined, black-and-white single-page site template for studios, agencies, and personal brands. Includes smooth Lenis scrolling, GSAP reveal animations, a full-screen loader, and a React/Next.js-ready TSX component.

## What's included
- `index.html` — standalone ready-to-use site
- `Lumina.tsx` — React/Next.js component version
- Accessible markup and responsive layout
- MIT license in `LICENSE`

## Quick start
- Open `index.html` directly
- Import `Lumina` in any React app

## Tech
- HTML + Tailwind CSS via CDN
- GSAP + ScrollTrigger
- Lenis smooth scroll
